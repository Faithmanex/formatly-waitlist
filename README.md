# Formatly Waitlist

The official waitlist landing page for **Formatly** - the intelligent academic formatting tool.

## Features

-   **High-Fidelity Design:** Glassmorphism UI with a focus on academic professionalism.
-   **Dark Mode Support:** Fully responsive dark/light theming.
-   **Supabase Integration:** Real-time email capture for the waitlist.
-   **Animate on Scroll:** Smooth reveal animations for a polished feel.

## Setup

1.  Clone the repository.
2.  Copy `.env.example` to `.env` (or set up your environment variables).
3.  Open `index.html` in your browser or serve via a local server (e.g., Live Server).

## Tech Stack

-   HTML5
-   CSS3 (Custom Properties for theming)
-   Vanilla JavaScript
-   Supabase (Backend)
